# 财流镜 FlowLens

> 家庭消费账单自动采集与管理桌面应用——一键抓取淘宝、京东、支付宝、微信、银行卡账单，本地 AI 自动分类，数据完全离线。

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 多平台账单采集 | 淘宝 / 天猫、京东、支付宝、微信支付、招商银行 |
| 家庭多成员管理 | 支持多个家庭，每个家庭下可添加多位成员 |
| 自动授权登录 | 内嵌悬浮浏览器，扫码 / 密码登录后自动保存凭证 |
| AI 智能分类 | 本地大模型对商品自动归入 17 大消费类别 |
| 数据完全本地 | 所有数据存入本机，不上传任何隐私信息 |

## 安装

前往 [Releases](../../releases) 页面下载对应系统的安装包：

- **macOS**：下载 `.dmg` 文件，打开后将应用拖入「应用程序」文件夹
- **Windows**：下载 `.exe` 安装包，按提示完成安装

## 快速上手

### 第一步：创建家庭

首次启动会弹出对话框，为家庭或组织命名（如「我家」），点击确认即可。

### 第二步：添加成员

点击左下角「**设置**」→「成员管理」→「**+ 添加成员**」，输入姓名（如：本人、配偶）。

### 第三步：绑定账号

选择成员 → 「账号管理」→ 选择平台（淘宝 / 京东 / 支付宝 / 微信 / 招商银行）→「**添加账号**」。

点击「**授权**」后，右下角弹出悬浮浏览器，在其中完成登录。登录成功后点击「✓ 完成授权」，系统自动识别并保存登录凭证。

### 第四步：采集账单

在左侧导航切换到对应平台，点击「**获取最近 3 个月数据**」，系统将自动：

1. 遍历所有已授权账号
2. 逐页抓取订单列表
3. 访问订单详情页提取完整字段
4. 调用本地 AI 对每件商品自动分类
5. 全部入库后弹出完成提示

## AI 分类（可选）

财流镜支持通过本地 [Ollama](https://ollama.com/) 对商品名称自动归类，分为数码3C、服饰鞋包、食品生鲜等 17 大类。

**开启步骤：**

1. 前往 [ollama.com](https://ollama.com/) 下载并安装 Ollama
2. 打开终端，拉取推荐模型：
   ```bash
   ollama pull qwen2.5:7b
   ```
3. 启动财流镜，在「设置 → AI 分类」中选择已下载的模型

> 若未安装 Ollama 或模型不可用，商品分类将自动标记为「其他」，不影响账单采集。

## 注意事项

- 本应用通过内嵌浏览器以**用户身份**访问各平台，与正常手动操作行为等同，请勿高频采集以避免账号风控
- 登录凭证（Cookie）仅存储在本机，不会上传至任何服务器
- 首次授权后 Cookie 有效期通常为 30 天，过期后需重新授权

---

## 数据存储位置

账单数据以 SQLite 格式存储在系统 userData 目录：

- **macOS**：`~/Library/Application Support/财流镜/flowlens.sqlite`
- **Windows**：`%APPDATA%\财流镜\flowlens.sqlite`
